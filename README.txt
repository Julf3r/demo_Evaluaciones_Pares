📌 Flujo de Datos

1️⃣ Python genera un archivo estudiantes.csv con la información de los estudiantes.
2️⃣ Excel importa el CSV y lo convierte en una tabla estructurada.
3️⃣ El estudiante ingresa su RUT en la hoja principal de Excel.
4️⃣ Un botón ejecuta un macro VBA que:

Filtra los datos según el RUT.

Muestra solo la información del estudiante en formato de tabla.

📂 Archivos

📄 main.py → Genera el CSV.📄 evaluación_pares.csv → Contiene los datos.📄 Tabla.xlsm → Archivo de Excel con macros.

🎯 Cómo Usarlo

1️⃣ Ejecutar main.py para generar evaluación_pares.csv.
2️⃣ Abrir Tabla.xlsm en Excel.
3️⃣ Ingresar el Nombre y presionar el botón para ver la información.
4️⃣Los nombres de prueba son: Juan, Jorge, Trinidad, Catalina.
¡Listo! 🎉
